Description & usage
==================
        h2bNh.py    ----tiny hex to bin and htxt(with crc creating)
            EX. h2bNh.py -f *.hex -s 0x3800
            
        raw2C       ----mxt raw to c file
            EX. raw2C -f *.raw
            
        bin2htxt     ----bin to c file only
            EX. bin2htxt f *.bin -s 0x3800
